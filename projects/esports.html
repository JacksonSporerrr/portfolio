Esports Performance Prediction Model

A self-directed machine learning project that scraped competitive Fortnite player data and used XGBoost pairwise ranking to forecast FNCS tournament outcomes.

Python XGBoost Selenium pandas scikit-learn

Overview

Competitive Fortnite (FNCS — Fortnite Champion Series) attracts thousands of players across global qualifiers and Grand Finals. Predicting team placement ahead of tournaments is a challenging problem — results depend on historical performance, current form, duo chemistry, and match-specific strategies like drop location. This self-directed project built an end-to-end pipeline: scraping public player stats, engineering features, training a ranking model, and generating tournament predictions.

Tools & Technologies

Python XGBoost Selenium pandas scikit-learn joblib CSV / Data Pipelines

Dataset

All data was self-collected from public web sources. Using Python scripts and Selenium for browser automation, I scraped:

  • Power Rankings (PR) — the competitive ranking metric used by the Fortnite community
  • Career earnings — total prize money won by each player
  • FNCS wins — number of major championship titles
  • Top 5 finishes — consistency indicator across major events
  • Session data — per-match stats (points, placements, eliminations, wins) across 15+ tournament sessions
  • Drop location quality — custom scoring of duo drop spots by zone safety and loot value

Process

  • Built Selenium and API-based scrapers to collect player stats from public Fortnite competitive tracking sites.
  • Cleaned and merged session-level match data with static player career stats into a unified feature set.
  • Engineered features per duo team: combined PR scores, earnings totals, FNCS win counts, Top 5 totals, average eliminations, and drop location quality.
  • Trained an XGBoost pairwise ranking model (rank:pairwise) on historical session data, treating lower placement rank as the target to maximize.
  • Applied Bonferroni-style model validation using held-out tournament sessions to evaluate prediction accuracy.
  • Generated ranked predictions for FNCS Heat sessions and Grand Finals using fncs_predict.py and combined_predict.py.

The model successfully ranked competitive duos and generated pre-tournament placement predictions for FNCS heats and Grand Finals. Drop location quality and Power Ranking emerged as the strongest predictive features. The pipeline was actively updated across multiple FNCS seasons, with new session data incorporated to refine predictions.

Business Insights

Competitive gaming organizations and esports analysts increasingly rely on data to scout talent, set betting lines, and understand team dynamics. This project demonstrates that public data — when properly cleaned and feature-engineered — can yield meaningful predictive signal even for fast-moving competitive environments. The same methodology could be adapted to other esports titles or used by talent agencies to identify undervalued players before they break out.
