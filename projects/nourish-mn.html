Nourish MN Food Shelf Database

A relational database system and barcode-based client tracking platform built to replace manual inventory management at Nourish MN, a Minnesota food shelf organization.

Python SQL JavaScript Relational Databases

Overview

Nourish MN operates a food shelf serving families across Minnesota. Their existing process relied on manual, paper-based tracking of food inventory and client visits — slow, error-prone, and difficult to report on. The goal of this project was to design and deploy a digital system that could handle inventory management, client tracking, and reporting in a unified platform.

Tools & Technologies

Python SQL JavaScript Relational Database Design Barcode Systems

Dataset

Since the organization had no prior digital records, we created structured test data to simulate real-world operations — including client households, inventory items, visit histories, and distribution logs. This allowed the system to be tested end-to-end before going live.

Process

  • Analyzed the organization's existing manual workflows to understand data requirements and pain points.
  • Designed a normalized relational database schema covering clients, inventory, visits, and reporting.
  • Built SQL queries and a Python backend to power automated reporting processes, replacing manual spreadsheet work.
  • Implemented a barcode-based client tracking system so staff could quickly log client visits and pull up household records at intake.
  • Developed a JavaScript front-end interface so non-technical staff could interact with the system without writing any SQL.

The digital database proved dramatically more efficient than the manual system — eliminating data entry errors, enabling instant inventory lookups, and automating reports that previously required hours of manual compilation.

Business Insights

Nonprofits often operate with limited staff and no dedicated IT resources. A well-designed database with a simple front-end interface can have an outsized impact — reducing administrative overhead, improving accuracy, and freeing staff to focus on serving clients rather than managing paperwork. Barcode integration brings warehouse-grade efficiency to community organizations at minimal cost.
